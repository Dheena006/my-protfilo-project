# 🌐 Personal Portfolio Website

This is a simple, clean, and responsive personal portfolio website that includes two main pages:

1. **Home Page**
2. **Contact Page**

The purpose of this project is to showcase my skills, projects, and provide a way for people to connect with me.

---

## 🏠 Home Page

The **Home Page** introduces who I am and what I do. It includes:

- A welcoming **intro section** with my name and title (e.g., "Hi, I'm Dheena R – Web Developer").
- A short **"About Me"** section describing my background and passion for development.
- A **skills section** listing the technologies I work with (e.g., HTML, CSS, JavaScript).
- A **project showcase** with images and links to live demos or GitHub repositories.
- Responsive design that works on desktop, tablet, and mobile devices.
- A simple and clean navigation bar.

---

## 📬 Contact Page

The **Contact Page** provides a way for visitors to get in touch. It includes:

- A working **contact form** with fields for name, email, and message.
- Basic form validation (e.g., required fields, valid email format).
- My **contact details** such as email and location.
- **Social media links** to GitHub, LinkedIn, etc.
- A success message after the form is submitted (if connected with a form service like EmailJS or Netlify Forms).

---

## 🚀 Technologies Used

- HTML5
- CSS3
- JavaScript (or React, if applicable)
